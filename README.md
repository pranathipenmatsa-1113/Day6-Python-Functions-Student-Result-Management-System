## 9. Temperature Converter

Converts

- Celsius → Fahrenheit
- Fahrenheit → Celsius

using separate functions.

### Formula Used

**Celsius to Fahrenheit**

```
(C × 9/5) + 32
```

**Fahrenheit to Celsius**

```
(F − 32) × 5/9
```

This project combines all the concepts learned throughout Day 6.

- Marks Entry for Five Subjects
- Total Marks Calculation
- Average Calculation
- Grade Assignment
- Pass/Fail Result
- Well Formatted Output

### Grade Calculation

| Average | Grade |
|----------|-------|
| 90+ | Grade A |
| 80–89 | Grade B |
| 70–79 | Grade C |
| 60–69 | Grade D |
| Below 60 | Grade E |

### Result Criteria

- PASS → All subjects have marks greater than or equal to 35.
